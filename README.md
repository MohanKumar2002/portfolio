# AI Researcher & Automation Engineer Portfolio

## ✨ Features
* **Modern & Clean Aesthetics**: Designed meticulously with a premium light-mode UI, utilizing glassmorphism, dynamic animations, and high-contrast typography.
* **Interactive Elements**: Features a 3D avatar that tracks the mouse pointer and an infinite-scrolling tech stack marquee.
* **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
* **Project Showcases**: Includes categorized sections for academic research (M.Tech, PhD, UG projects) and full-stack AI platform deliveries.

## 🛠️ Tech Stack
This project was primarily built with the following technologies:
* **Framework**: React 18 + Vite
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion
* **Icons**: Lucide React

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MohanKumar2002/portfolio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd portfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to `http://localhost:5173/` to view the application.
